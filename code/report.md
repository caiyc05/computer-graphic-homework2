## report.md
1. 代码逻辑
	1. 实现object3d子类的intersect函数：
		- sphere采用几何求交方法
		- plain 也采用几何求交的方法
		- triangle先采用plain的几何求交的算法，再使用重心法判断是不是在三角形中
		- 每一次的intersect都要判断是否满足大于$t_{min}$,并且离camera最近，也就是hit中的t是最小的
	2. 实现group类：
		- 使用vector将所有的object存储起来
		- 对整个group执行intersect的时候所有的vector中的object都执行一次intersect
	3. 实现perspectiveCamera类的构造函数和generateRay方法
		- 基类 Camera，存储相机位置、朝向、上下左右方向、图像宽高
		- 构造函数自动计算正交坐标系：前、右、上
		- 子类 PerspectiveCamera，根据视角 angle 计算焦距 fx, fy
		- generateRay 生成射线逻辑，把像素 `(u, v)` 转到相机中心坐标系
		- 计算相机空间射线方向 `(x,y,1)` 并归一化
		- 用旋转矩阵 `R = [horizontal, -up, direction]`
   		- 把方向转到世界空间
		- 返回一条起点在相机、方向计算好的射线
	4. main.cpp 的实现逻辑
		- 读取命令行输入的场景文件和输出图片路径
		- 用 `SceneParser` 解析场景，得到相机、物体、光源
		- 遍历图像每个像素 (x,y)
		- 调用相机生成一条射线，射向当前像素
		- 检测射线与场景物体是否相交
		- 有交点：遍历所有光源，计算光照颜色，赋值给像素
		- 无交点：使用背景色
		- 所有像素计算完成，保存成图片
	5. 实现Material类的Shade方法
		- 获取法线、光线方向、视线方向
		- 计算反射方向 R
		- 计算漫反射分量
		- 计算镜面反射分量（视线×反射点积，加光泽度指数）
		- 漫反射 + 镜面反射 → 叠加得到最终颜色
		- 返回着色结果
